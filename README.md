# DevOps101

A hands-on, no-fluff path through the foundations of DevOps — Linux, AWS, Terraform, and Docker. Each module has concept notes followed by labs you actually run.

## Modules

1. **[Linux](./Linux/)** — filesystem, permissions, processes, networking, scripting, logs.
2. **[AWS](./AWS/)** — CLI, EC2, VPC, IAM, S3, RDS.
3. **[Terraform](./Terraform/)** — HCL, state, variables, modules, debugging.
4. **[Docker](./Docker/)** — images, containers, Dockerfiles, volumes, networks, Compose, best practices.

## How can you start Right Away?

1. https://distrosea.com/ — Quickest but limited
2. AWS or any cloud free tier
3. Virtual machine on your local — VirtualBox or VMware
4. Docker Desktop — for the Docker module, and for trying Linux without a VM
